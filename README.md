#### 自定义播放器

subNVues 是 vue 页面的原生子窗体。用于解决App中 vue 页面中的层级覆盖和原生界面灵活自定义用的。
它是一个 nvue 页面，使用 weex 引擎渲染，方便自定义原生导航或覆盖原生地图、视频等
虽然有cover-view来覆盖原生组件，但是在app-vue中不支持嵌套，且只能在map、video组件使用，这个时候用原生子窗体是比较适合的
